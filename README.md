# vue-crash-2021

Vue 3 project by Traversy Media.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Bundles and serves production build
```
npm run deploy
```

### Start backend server
```
npm run backend
```
